Digitisation tutorial

Hello new SDR user! This tutorial will take you through the basic process of using the SDR: * Uploading an input file to the SDR * Launching the workflow to perform the digitisation * Viewing and saving the digitisation output

Getting ready

For this tutorial, please download and use the following reference file. If you have already following this tutorial and are interested in creating your own input files, please see this guide to creating your own input files.

Uploading input file

The SDR is driven by supplying an input file. This file contains some basic details about the sample to be digitised and, crucially, a URL pointing to the specimen image itself. Each column of the file corresponds to an element of metadata and each row of the file is a sample to be analysed.

To allow the SDR to process our reference file, we must upload it to the SDR interface.
